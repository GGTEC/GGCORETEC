Comandos !

Esses são os comandos atualmente ativos no canal.

Comandos publicos ou que são necessários niveis de permissão superiores (vip/sub)

Comando Descrição
!nick Exibe os nicks dos principais jogos do canal
!discord !dc Envia um link de convite para o discord do canal
!lurk Envia uma mensagem avisando que vai deixar no lurk.
!meta Exibe a meta atual do canal na tela
!elo Exibe o elo e outras informações do league of legends
!kdr Exibe o kdr do valorant
!sub Exibe o site da livepix para enviar um sub
!pix Exibe os links para o livepix
!comandos Exibe o link para esta pagina
!social Exibe o link para esta pagina
!bot Exibe o link para o site do bot
!smurf Exibe o nickname da conta smurf do valorant
!rabo (user) Brincadeira que mostra quanto de fogo no rabo o usuário está
!atual Exibe a musica tocando (se tiver tocando uma)
!proxima Exibe a proxima musica para ser tocada (se tiver tocando uma)
!skip Pula para a proxima musica
!clip Exibe a proxima musica para ser tocada (se tiver tocando uma)
!dado Escolhe um número entre 1 e 6
!random (numero) Escolhe um numero a partir do valor digitado
!followage Exibe o tempo que você segue o canal
!game Exibe o nome do game
!selfcheck Verifica se o usuário está no sorteio
!uptime Exibe a duração atual da live
!msgcount Exibe a quantidade de mensagens recebida pelo bot vindas do usuário
!tempo Exibe o tempo que o bot detectou que o usuário assitiu a live
!tapa (user) Mensagem interativa para tapa em um usuário
!beijo (user) Mensagem interativa para beijo em um usuário
!abraço (user) Mensagem interativa para abraço em um usuário
!voadora (user) Mensagem interativa para voadora em um usuário
!socão (user) Mensagem interativa para socão em um usuário
!duelo (user) Envia uma solicitação de duelo para um usuário
!duelo aceitar Aceita uma solicitação de duelo de um usuário
!cmd create (comando) (resposta) Cria um comando no chat com uma resposta simples, (necessário utilizar parenteses e prefix "!,@,#..")
!cmd edit (comando) (resposta) Edita o comando, devem ser seguidas as mesmas regras anteriores
!cmd remove (comando) Remove o comando
!accountage (user) Exibe o tempo que a conta foi criada
!sortear Executa um sorteio em andamento
!limpar_sorteio Limpa a lista de usuários no sorteio
!check_user (user) Verifica se o usuário está no sorteio
!adduser : Adiciona usuário na lista de sorteio
