Substituição do NTFS pelo ReFS no Windows 11

Windows OS Noticias

O Windows 11 é o novo sistema operacional da Microsoft que promete trazer mais segurança, desempenho e recursos para os usuários. Uma das novidades que pode chegar em breve é a substituição do sistema de arquivos NTFS pelo ReFS, que oferece mais resiliência contra corrupção de dados e outras vantagens.

O que é o ReFS?

O ReFS (Resilient File System) é um sistema de arquivos desenvolvido pela Microsoft para superar as limitações do NTFS (New Technology File System), que é usado desde o Windows NT 3.1. O ReFS foi introduzido pela primeira vez na versão de servidor do Windows 8.1 e estava disponível apenas para algumas edições do Windows 10.

O ReFS tem como objetivo oferecer mais confiabilidade, escalabilidade e eficiência para o armazenamento de dados. Algumas das características do ReFS são:

  • - Proteção contra corrupção de dados: o ReFS usa um mecanismo chamado "integridade de metadados" que detecta e corrige erros nos metadados (informações sobre os arquivos) sem afetar os dados em si. Além disso, o ReFS pode usar a tecnologia Storage Spaces para criar pools de armazenamento com espelhamento ou paridade, que podem recuperar dados em caso de falha de um disco.
  • - Otimização para grandes volumes: o ReFS pode lidar com volumes de até 1 yottabyte (10^24 bytes) e arquivos de até 16 exabytes (10^18 bytes), superando os limites do NTFS. O ReFS também usa um sistema de alocação dinâmica que reduz a fragmentação e melhora o desempenho.
  • - Suporte a snapshots: o ReFS permite criar snapshots (cópias instantâneas) dos volumes, que podem ser usados para restaurar dados em um estado anterior ou para fazer backup. Os snapshots são criados usando a tecnologia Volume Shadow Copy Service (VSS) e podem ser gerenciados pelo PowerShell ou pelo Gerenciador de Servidor.
  • - Compatibilidade com o NTFS: o ReFS mantém a compatibilidade com a maioria dos recursos do NTFS, como permissões, criptografia, compactação e links simbólicos. No entanto, alguns recursos não são suportados pelo ReFS, como deduplicação de dados, transações e pontos de restauração.

Como usar o ReFS no Windows 11?

Atualmente, as edições de consumidor do Windows 11 não permitem instalar o sistema operacional no ReFS, mas apenas formatar discos externos ou secundários com esse sistema de arquivos. No entanto, há indícios de que a Microsoft está trabalhando para tornar o ReFS o sistema de arquivos padrão nas novas instalações do Windows 11.

Um usuário do Twitter descobriu que é possível ativar uma opção oculta (ID 42189933) na última prévia do Windows 11 disponível no Dev Channel para instalar uma cópia limpa do Windows 11 usando ReFS. Segundo ele, a instalação não apresentou problemas e o Windows 11 funcionou normalmente. Não se sabe quando a Microsoft vai liberar essa opção para todos os usuários, mas é provável que isso aconteça em breve, já que o Windows 11 pode inicializar a partir de uma partição ReFS sem dificuldades. Essa mudança pode trazer mais segurança e robustez para o sistema operacional, além de aproveitar melhor os recursos dos discos modernos.

Conclusão

​O ReFS é um sistema de arquivos que pode substituir o NTFS nas novas instalações do Windows 11, trazendo mais resiliência contra corrupção de dados e outras vantagens. O ReFS já está presente na versão de servidor do Windows e pode ser ativado manualmente na versão desktop. A Microsoft está trabalhando para tornar o ReFS o sistema de arquivos padrão no Windows 11 em breve.
