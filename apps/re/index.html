Instruções de Download

Passo 1: Acesse o Link

Acesse o seguinte link para efetuar o download do programa: https://github.com/GGTEC/RewardEvents/releases

Passo 2: Baixe o Arquivo

Baixe o arquivo "RewardEvents.zip".

Passo 3: Efetue uma Atualização (opcional)

Se estiver efetuando uma atualização, na maioria dos casos, é necessário apenas baixar o arquivo "RewardEvents.exe".

Passo 4: Armazenamento

Não é necessário instalação. Armazene o programa em uma pasta separada para melhor organização.

Autenticação de Conta Streamer

  1. Após clicar em "Login", a tela principal do aplicativo ficará congelada e somente a tela do navegador estará funcionando.
  2. Digite o nome da sua conta de streamer no campo especificado. Ele deve ser o mesmo que o usado na URL, por exemplo, https://www.twitch.tv/gg_tec = gg_tec.
  3. Clique em "Login".
  4. Faça login na sua conta e autorize o aplicativo a se comunicar com sua conta e obter as informações necessárias para o funcionamento. As permissões requeridas serão exibidas no momento em que a autorização for solicitada.

Autenticação de Conta Bot

Aviso Importante: É altamente recomendado o uso de uma conta Bot para evitar problemas de conexão com o chat durante a inicialização do programa.
  1. Se você não deseja usar uma conta Bot, clique no botão indicado.
  2. Digite o nome de usuário da sua conta Bot no campo correspondente. Deve ser o mesmo que a URL, por exemplo, https://www.twitch.tv/gg_tec = gg_tec.
  3. Clique em "Login".
  4. Faça login na sua conta e autorize o aplicativo a se comunicar com sua conta e coletar as informações necessárias para o funcionamento correto. As permissões requeridas serão exibidas na tela quando a autorização for solicitada.

Conexão OBS Studio

Para estabelecer a comunicação com o OBS Studio e modificar as informações, é necessário ter a versão 28+ do OBS Studio instalada, versões anteriores não são compatíveis.

  1. Abra o OBS Studio.
  2. Vá até a aba "Ferramentas" e clique em "Configurações OBS-WebSocket".
  3. Marque a caixa "Ativar Servidor WebSockets".
  4. Em "Porta de Servidor", teste uma porta que funcione, por exemplo, 4444.
  5. Marque a caixa "Habilitar Autenticação".
  6. Em "Senha de Servidor", escolha uma senha de sua escolha, por exemplo, 1234.
  7. Aplique e salve as configurações.
  8. No RewardEvents, vá até as Configurações e clique em "Conexão com o OBS Studio".
  9. Utilize os mesmos valores configurados no OBS Studio nos campos Porta e Senha no RewardEvents.
  10. Em "Host", utilize "localhost".
  11. Clique em "OK" e a configuração inicial estará salva.

Configurações de Notificações no OBS Studio

Siga estes passos para configurar as notificações no OBS Studio:

  1. Vá até a seção "Configurações" e selecione "Notificações OBS Studio".
  2. Marque a opção "Ativar notificações para resgate de recompensas no OBS Studio" para exibir uma notificação no OBS Studio quando uma recompensa for resgatada.
  3. Marque a opção "Ativar notificações para pedidos de música no OBS Studio" para exibir uma notificação no OBS Studio quando uma música for reproduzida.
  4. Crie uma fonte de navegador no OBS Studio e configure-a para usar o arquivo HTML de notificações na pasta "web/src/html" do RewardEvents. Selecione a opção "Arquivo local" e navegue até a pasta onde o arquivo HTML está localizado. Configure a largura como 768 pixels e a altura como 1360 pixels. Redimensione a fonte conforme sua preferência.
  5. Em "Tempo de exibição da notificação", coloque o tempo em segundos que a notificação deve ficar visível na tela.

HTML para notificações

Ativar notificações via HTML

Ative se deseja exibir os alertas no OBS Studio

Configure uma nova fonte de navegador no OBS Studio, ative a opção "Arquivo local", a localização do arquivo é "web/src/html/event.html" localizada na pasta web do RewardEvents

Selecione a fonte que está configurada o arquvio HTML de Notificações

Clique no botão carregar, no seletor estará listada todas as fontes de cena atual no OBS studio, selecione a fonte de navegador criada anteriormente que foi configurada com o arquivo HTML para alertas.
