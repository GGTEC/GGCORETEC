Guia de configurações

Configurações necessárias para o funcionamento correto do programa

  • Autenticar conta streamer

    1. Depois de clicar em login o aplicativo deve ficar com a tela principal congelada e somente a tela de navegador funcionando.

    2. Digite o nome da sua conta de streamer no campo requisitado. (deve ser igual o da url ex. https://www.twitch.tv/gg_tec = gg_tec )

    3. Clique em login

    4. Logue na sua conta e autorize o aplicativo para comunicar com a sua conta e coletar as informações necessárias para o funcionamento do aplicativo, as permissões coletadas são exibidas no momento em que é requisitada a autorização.

  • Autenticar conta bot

    1. Caso não queira usar a uma conta Bot clique no botão indicado.

    2. Digite o nome de login da sua conta bot no campo requisitado. (deve ser igual o da url ex. https://www.twitch.tv/gg_tec = gg_tec )

    3. Clique em login

    4. Logue na sua conta e autorize o aplicativo para comunicar com a sua conta e coletar as informações necessárias para o funcionamento do aplicativo, as permissões coletadas são exibidas no momento em que é requisitada a autorização.

  • Configurações : Conexão OBS Studio

    1. Para comunicarmos com o OBS e modificar as informações para isso precisamos instalar a ferramenta OBS-WEBSOCKET Compat

    2. Baixe e Instale o OBS-WEBSOCKET Compat

    3. Abra o OBS studio, vá até a aba ferramentas e clique em WebSockets Server Settings

    4. Marque a caixa Enable WebSockets server;

    5. Em serve port teste alguma que funcionar ex. 4444;

    6. Marque a caixa enable authentication;

    7. Em password use a senha que preferir ex. 1234;

    8. Utilize os mesmos valores no campo port e password no programa;

    9. Em Host utilize localhost;

    10. Clique em OK e a configuração inicial estará salva;

    Se preferir que o programa conecte automaticamente com o OBS Studio (se estiver sendo executado) na inicialização marque a opção correspondente

    Na aba perfil é possivel conectar manualmente o OBS Studio

  • Configurações : Widget OBS Studio

    1. Marque a opção 'Ativar notificalções HTML';

    2. Em 'Fonte HTML OBS' selecione a fonte de navegador que foi configurado o HTML para notificações que está na pasta src/html;

    3. Em 'tempo exibindo a notificação' coloque o tempo em segundos que a notificação deveria ficar visivel na tela;

  • Configurações : Comandos

    Configurar comandos

    Prefixo : Qual prefixo vai vir antes do comando ex= !, ?, $, # ex = !comando, ?comando $comando, #comando;

    Pedido de musica : Qual será o comando para pedidos de musica (somente moderador) uso = !sr musica;

    Musica atual : Qual será o comando para o usuário saber qual é a musica que está tocando no momento;

    Próxima musica : Qual será o comando para saber qual será a próxima musica na fila;

    Volume : Qual comando será utilizado para ajustar o volume da musica (somente moderador) uso = !volume 50;

    Pular musica ? :Qual comando será utilizado para pular a musica (somente moderador);

    Delay entre comandos: Quanto tempo o usuário deverá esperar antes de usar qualquer comando.

  • Configurações : Recompensa de pedidos

    Configure a recompensa de pedidos

    Recompensa : Selecione a recompensa que receberá pedidos de musica, a recompensa deve ter a opção 'Exigir que um usuário insira um texto' para receber o nome da musica;

  • Configurações : Mensagens no chat

    Configure quais mensagens/modulos devem ser habilitados

    Habilitar comando : Ativa o uso de comandos no chat

    Ativar respostas de comandos : Ativa a resposta dos comandos

    Exibir erro de permissão para comandos : Ativa a resposta de erro quando um usuário tentar utilizar um comando que é só para moderadores

    Status de conexão no chat : Ativa o envio de status de conexão dos sistemas para receber e enviar mensagens no chat

    Enviar nome de musicas no chat : Ativa o envio de nome da musica quando iniciar a reprodução

    Confirmações de pedidos no chat : Ativa o envio de resposta de pedidos quando uma nova musica for adicionada na fila

    Erros de pedidos no chat : Ativa o envio de uma mensagem de erro quando uma musica ou link não puder ser processado

    Exibir delay para comandos : Ativa a resposta do delay para usar comandos se um comando for utilizado antes do tempo

  • Playlist

    Adicionar uma playlist inteira para ser reproduzida

    Url da playlist : Insira a URL da playlist do youtube, o formato do link deve ser esse = https://www.youtube.com/watch?v=U0CGsw6h60k&list=PLMC9KNkIncKtPzgY-5rmhvj7fax8fdxoj

    Executar playlist : Executa a playlist se estiver musicas na fila

    Limpar playlist : Limpa as musicas restantes na fila de playlist

    Musicas na playlist : Exibe uma lista das musicas na playlist

    Lista de pedidos

    É possível visualizar uma lista com as musicas solicitadas via chat ou resgate da recompensa configurada

  • Configurações : Editar respostas

    Configure as respostas padrão do programa

    Resposta : Selecione o tipo de resposta para editar

    Nota: Os prefix suportados já estão nas mensagens, nenhum prefix diferente dos quais já contem nas mensagens deve ser utilizado nas mensagens padrão.
