Guia de configurações

Crie eventos ou configure o programa

  • Autenticar conta streamer

    1. Depois de clicar em login o aplicativo deve ficar com a tela principal congelada e somente a tela de navegador funcionando.

    2. Digite o nome da sua conta de streamer no campo requisitado. (deve ser igual o da url ex. https://www.twitch.tv/gg_tec = gg_tec )

    3. Clique em login

    4. Logue na sua conta e autorize o aplicativo para comunicar com a sua conta e coletar as informações necessárias para o funcionamento do aplicativo, as permissões coletadas são exibidas no momento em que é requisitada a autorização.

  • Autenticar conta bot

    1. Caso não queira usar a uma conta Bot clique no botão indicado.

    2. Digite o nome de login da sua conta bot no campo requisitado. (deve ser igual o da url ex. https://www.twitch.tv/gg_tec = gg_tec )

    3. Clique em login

    4. Logue na sua conta e autorize o aplicativo para comunicar com a sua conta e coletar as informações necessárias para o funcionamento do aplicativo, as permissões coletadas são exibidas no momento em que é requisitada a autorização.

  • Conexão OBS Studio

    1. Para comunicarmos com o OBS e modificar as informações para isso precisamos instalar a ferramenta OBS-WEBSOCKET Compat

    2. Baixe e Instale o OBS-WEBSOCKET Compat

    3. Abra o OBS studio, vá até a aba ferramentas e clique em WebSockets Server Settings

    4. Marque a caixa Enable WebSockets server;

    5. Em serve port teste alguma que funcionar ex. 4444;

    6. Marque a caixa enable authentication;

    7. Em password use a senha que preferir ex. 1234;

    8. Utilize os mesmos valores no campo port e password no programa;

    9. Em Host utilize localhost;

    10. Clique em OK e a configuração inicial estará salva;

    Se preferir que o programa conecte automaticamente com o OBS Studio (se estiver sendo executado) na inicialização marque a opção correspondente

    Na aba perfil é possivel conectar manualmente o OBS Studio

  • Notificações OBS Studio

    1. Marque a opção 'Ativar notificalções HTML';

    2. Em 'Fonte HTML OBS' selecione a fonte de navegador que foi configurado o HTML para notificações que está na pasta src/html;

    3. Em 'tempo exibindo a notificação' coloque o tempo em segundos que a notificação deveria ficar visivel na tela;

  • Configurando os eventos de acordo com as recompensas

  • Configurações : Mensagens no chat

    Configure quais mensagens/modulos devem ser habilitados

    Habilitar comando !tts : Ativa o uso do comando para o evento 'Texto falado google' no chat

    Habilitar comando : Ativa o uso de todos os comandos no chat

    Ativar respostas de comandos : Ativa a resposta dos comandos simples e comandos de eventos

    Exibir delay para comandos : Ativa a resposta do delay para usar comandos se um comando for utilizado antes do tempo

    Exibir confirmações/erros de clipes : Ativa a resposta quando um clipe for criado ou ocorrer um erro

    Exibir erro de permissão para comandos : Ativa a resposta de erro quando um usuário tentar utilizar um comando que é só para moderadores

    Ativar timer : Ativa o modulo de mensagens automaticas

    Status de conexão no chat : Ativa o envio de status de conexão dos sistemas para receber e enviar mensagens no chat

  • Configurações : Sorteio

    Configure os comandos e o sistema de sorteios

    Criar um novo sorteio

    Nome do sorteio : Insira o nome que será anunciado no chat quando o sorteio for habilitado

    Adicionar usuário no sorteio

    Nome do usuário : Insira o nome do usuário que será adicionado manualmente no sorteio (não é necessário ser um nickname da twitch)

    Ativar sorteio : Permite a entrada de usuários no sorteio e envia uma mensagem no chat anunciando que o sorteio foi habilitado

    Limpar os nomes quando sortear : Quando o sorteio for executado os nomes da lista são limpos, manter essa opção for desmarcada é util para mostrar os participantes caso tenha alguma objeção ou reclamação sobre o sorteio, também é guardado um arquivo de backup dos nomes sorteados na pasta src/sorteio/

    Limpar nomes : Limpa os nomes da lista do sorteio antes de executar.

    Exibir nomes : Exibe uma lista com os nomes dos participantes do sorteio

    Executar o sorteio : Executa o sorteio

    Configurar comandos

    Executar sorteioQual comando deve ser enviado para executar o sorteio (necessário '!')

    Limpar nomesQual comando deve ser enviado para limpar nomes da lista do sorteio (necessário '!')

    Verificar usuárioQual comando deve ser enviado para verificar se o usuário está no sorteio (necessário '!')

    Usuário verificar participaçãoQual comando deve ser enviado para o usuário verificar a própria participação no sorteio (necessário '!')
